# SOP — Daily Reconciliation

**Audience:** Supervisor + Stores (+ Finance if in pilot)  
**When:** End of each pilot production day

## Purpose

Prove the day: WOs progressed, stock moved, QC done — without trusting mock dashboards.

## Checklist

1. **WO list:** Released / running / closed counts vs plan sheet.  
2. **Progress:** Sample 2–3 WOs — My Work + Daily Update totals vs physical output.  
3. **Materials:** Issue/return document numbers vs stores log; spot-check one item on-hand via API (not demo SPA).  
4. **Quality:** Queue cleared or exceptions listed; no incoming ERP QC claimed.  
5. **Corrections:** List any corrections with reasons.  
6. **Prohibited screens:** Confirm team did not use MRP / GRN QC / costing / budgeting / pick-pack.  
7. **Finance (optional):** Journals / Money In-Out only on canonical routes.

## Evidence

- Dated recon sheet (paper or shared doc) with WO numbers and movement ids.  
- Escalate mismatches same day; do not leave overnight without note.

## Do not

- Sign off based on live-activity chrome or manufacturing costing KPIs.
